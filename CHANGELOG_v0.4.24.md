# 工作量统计工具 v0.4.24 更新说明

## 变更概述
新增「事业部」层级（高于实验室），分析检测与研发送样双线同步支持事业部维度。实验室选择页加入事业部 Chip 筛选按钮（类似"液相/气相/理化"）。

## 新增功能

### 1. 事业部主数据
- 新增 `divisions` 表，预置 8 类事业部：液相、气相、理化、ICP、热分析、质谱、红外、其他
- 管理页新增「事业部管理」卡片，支持增删改
- 删除事业部时**级联置空**：下属实验室解除关联（division_id 置 NULL）但实验室本身保留

### 2. 实验室归属事业部
- `project_groups` 增加 `division_id`，实验室 N:1 归属某事业部
- 管理页实验室编辑弹窗新增「所属事业部」下拉（含"未分配"项）
- 现有实验室默认未分配，由管理员在管理页手动指派

### 3. 实验室选择页事业部筛选
- 工作量录入（分析检测 / 研发送样）的实验室选择页新增事业部 Chip 筛选栏
- 「全部 (n)」显示当前搜索条件下实验室总数，各事业部 Chip 显示其下属实验室数
- 点击事业部 Chip 过滤实验室卡片（复用现有 Chip 样式）

### 4. 记录冗余事业部
- `work_records` / `rd_work_records` 增加 `division_id`，录入时由所属实验室锁定写入
- 历史统计不受实验室后续改属影响

## 数据库变更
- 新建 `divisions` 表（id / name UNIQUE / sort_order / color / is_active / deleted_at / created_at）+ 8 类种子
- `project_groups` / `work_records` / `rd_work_records` 各增加 `division_id` 列 + 索引
- 旧数据 `division_id` 为 NULL（前端显示"未分配事业部"），不自动回填

## 版本隔离
- 本版本在独立文件夹 `v0.4.24` 实现，`v0.4.21` / `v0.4.22` / `v0.4.23` 源码均未改动

## 暂不含（留待后续版本）
- 统计 by_division 汇总端点
- 导出事业部列 / 独立事业部汇总 Sheet
- 事业部颜色显示 / 软删除
（数据模型已预留 color / is_active / deleted_at 列，后续可直接启用）

## 打包
- 安装包支持覆盖安装上一版本（AppId / AppName / DefaultDirName 与历史版本一致）
