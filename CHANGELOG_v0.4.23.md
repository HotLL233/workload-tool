# 工作量统计工具 v0.4.23 更新说明

> 发布日期：2026-07-09
> 基于 v0.4.22，对「样品信息登记」模块进行增强。

## 一、新增功能

### 1. 检测时间自动生成
- 登记表单**不再手动填写**「检测时间」。
- 当记录状态流转为「检测完成」时，后端自动写入检测时间（`date('now','localtime')`）。
- 列表 / 详情页只读展示检测时间。

### 2. 管理页新增「样品信息登记管理」卡片
- **检测类型 CRUD**：新增 / 编辑 / 软删除检测类型（名称、描述、颜色、排序）。
- **记录查询**：支持按 时间 / 送样人 / 实验室 / 项目 / 类型 / 状态 **全维度组合筛选**。
- **独立统计**：按 状态 / 类型 / 实验室 / 项目 / 送样人 / 月份 聚合（**独立于**分析检测统计，不接 `/stats`）。

### 3. 门户卡片自定义
- 门户首页的检测类型卡片（原为硬编码 ICP / 热分析 / 质谱 / 其他）改为从数据库 `sample_info_types` 表读取，可在管理页自由增删改。

### 4. 独立导出模块
- 新增「样品信息登记」专用导出，共 **7 个 Sheet**（明细 / 状态 / 类型 / 实验室 / 项目 / 送样人 / 月份）。
- 模板风格同分析检测导出，但内容完全独立，独立接口，不接 `/stats`。

## 二、需求落地核对（确认的 5 点）

| # | 需求 | 落地方式 |
|---|------|---------|
| ① | 软删除 | 检测类型使用 `is_active=0`；记录沿用 v0.4.22 既有 `deleted_at` 软删除机制（均为软删除，未物理删除） |
| ② | 统计维度按分析检测现有分类 | 按 状态 / 类型 / 实验室 / 项目 / 送样人 / 月份 聚合，复刻 `stats_handler` 写法且完全独立 |
| ③ | 全维度筛选 | 记录查询含 时间 / 人员 / 实验室 / 项目 / 类型 / 状态 |
| ④ | 颜色自由填色值 | `sample_info_types.color` 存色值字符串（如 `#2e7d32`），表单自由输入 |
| ⑤ | 需导出且独立 | 独立 7-Sheet 导出，独立接口 |

## 三、版本隔离
- 本版本位于**独立文件夹 `v0.4.23`**（从 `v0.4.22` 复制），全程未改动 `v0.4.21` / `v0.4.22`。

## 四、已知限制
- 记录删除沿用 v0.4.22 的 `deleted_at` 软删除（与检测类型的 `is_active=0` 列名不同，但语义一致，均为软删除）。
- 历史数据若没有 `type_key`，将按 `detection_type` 字段兜底展示。
