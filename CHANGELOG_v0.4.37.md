# v0.4.37 更新说明

> **版本**: 0.4.37 | **日期**: 2026-07-11 | **基线**: v0.4.36

## 全站 UI 可视化编辑系统

### 新增功能
- **13 个页面全部支持可视化编辑**：管理员登录后，任意页面右上角点「✎ 编辑布局」进入编辑模式
- **PageSectionEditor**：每个页面被拆分为多个 section，可独立控制显隐和标题
- **编辑模式**：编辑 label 文本（标题/按钮文字）、切换 section 显示/隐藏、实时保存
- **功能零影响**：非编辑模式完全透明，原功能不受任何影响

### 可编辑的页面和 section

| 页面 | 可编辑 section |
|------|---------------|
| 主页 | 品牌标题、入口卡片区、底部文案 |
| 研发送样录入 | 页面标题、表格列、提交/重置按钮 |
| 研发送样门户 | 页面标题、实验室过滤、实验室卡片 |
| 分析检测门户 | 页面标题、实验室过滤、实验室卡片 |
| 样品信息登记 | 页面标题、表单字段、提交按钮 |
| 样品信息首页 | 页面标题、类型卡片 |
| 研发送样记录 | 页面标题、表格列、过滤器 |
| 分析检测统计 | 页面标题、统计卡片、图表 |
| 研发送样统计 | 页面标题、统计卡片、图表 |
| 检测录入 | 页面标题、表单字段、提交按钮 |
| 样品信息统计 | 页面标题、统计卡片、图表 |

## 技术细节
- **新建文件**（1 个）：`PageSectionEditor.tsx`
- **修改文件**（12 个）：`migrations.rs`、10 个页面组件 + 种子数据
- **版本隔离**：基于 v0.4.36 创建独立文件夹 v0.4.37

## 升级说明
- 安装后首次启动自动插入 10 条 layout 种子数据
- 管理员登录后，任意页面上方出现「✎ 编辑布局」按钮
- 安装包可覆盖安装 v0.4.30~v0.4.36（AppId 固定）
