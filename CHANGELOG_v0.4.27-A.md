# v0.4.27-A 更新说明

> **版本**: 0.4.27-A（特别版） | **日期**: 2026-07-10

## 新增功能

### 1. 列自定义绑定检测类型（需求2）
- 新增 `sample_info_column_visibility` 桥接表，每个检测类型独立控制预置列的显示/隐藏
- 预置列全局共享但可按类型差异化：管理页选中类型后，每条预置列有 Switch 开关控制在当前类型的显隐
- 自定义列仅属于当前选中的检测类型
- 创建新检测类型时自动初始化 12 条预置列的可见性记录
- 样品登记录入、记录列表、导出均按类型加载对应列配置

### 2. 附件上传（需求4）
- 样品信息登记记录支持上传附件（PDF / Word）
- 上传后存储为 `seq_{序号}_{记录ID}_{时间戳}_{原始文件名}`
- PDF 点击在新标签页浏览器预览，Word 触发下载
- 附件元数据存 `sample_info_attachments` 表，文件存 `data/attachments/` 目录

### 3. 研发送样多行表格（需求5）
- 研发送样录入页改为 Excel 式多行表格
- 外层卡片入口（SamplePortal）完全保留
- 多行表格列：序号/送样人/部门/实验室/项目/类型/方法/数量/批号/注意事项
- 支持添加行、删除选中行、重置、批量提交
- `rd_work_records` 新增 `batch_no`（样品批号）、`notes`（注意事项）字段

### 4. 部门关联实验室（需求3）
- 部门编辑弹窗新增「关联实验室」复选框列表
- 复用现有 `project_groups.division_id`，通过批量更新 API 同步

### 5. 用户管理 + 登录系统（需求6）
- 新增用户注册/登录/登出功能（JWT + bcrypt）
- 管理员账号 `admin/admin123` 启动时自动创建
- 用户管理卡片（管理页新增 Tab）：用户表格含权限列、编辑弹窗支持管理员开关
- 登录后：研发送样卡片页自动筛选用户部门并置顶用户实验室
- 登录后：录入手动填充送样人/部门/实验室（可手动修改）
- 路由守卫：研发送样和样品信息登记入口需登录

### 6. 移除统计概览（需求1）
- 管理页样品登记 Tab 移除顶部「记录统计概览」卡片
